# 🏦 Banking System Simulator

A simple, interactive Java console application that allows users to manage their bank balance through a menu-driven interface.

## ✨ Features
* **Deposit:** Add funds to your balance.
* **Withdraw:** Securely remove funds with balance validation.
* **Check Balance:** Instant view of current funds.
* **Infinite Loop Menu:** Keeps the app running until the user chooses to exit.

## 💻 Technical Skills Shown
* **Java Fundamentals:** Variables, Scanner, and Data Types.
* **Logic:** `if-else` branching and `while(true)` loops.
* **Code Structure:** Modular functions (methods) for clean code.
